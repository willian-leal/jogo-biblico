# Jogo Bíblico

Este repositório contém a base inicial de uma aplicação composta por:

- **Backend** em C# usando .NET 6 com uma API mínima.
- **Frontend** em Angular com uma estrutura simplificada.

## Estrutura

- `backend/JogoBiblicoBackend/` - Projeto ASP.NET 6.
- `frontend/jogo-biblico/` - Aplicação Angular.

## Como executar

Devido às limitações do ambiente do repositório, não há scripts de build automáticos. Para compilar e executar cada parte é necessário ter o SDK do .NET e o Node com o Angular CLI instalados em seu ambiente local.
